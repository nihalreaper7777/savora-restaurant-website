# Savora Restaurant Website

A modern and elegant restaurant landing page built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project showcases a premium Italian fine dining website with a responsive layout, interactive menu filtering, and reservation form validation.

---

## 🌐 Live Demo

**Website:**
https://nihalreaper7777.github.io/savora-restaurant-website/

---

## ✨ Features

* 🎥 Full-screen video hero section
* 🍝 Premium restaurant-themed user interface
* 📱 Responsive design
* 🧭 Smooth scrolling navigation
* 🍽️ Interactive menu filtering using JavaScript
* 🖼️ Gallery section
* 📅 Reservation form with validation
* 📍 Visit Us section
* 📩 Modern footer
* 🎨 Hover effects and transitions
